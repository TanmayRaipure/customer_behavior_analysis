# Customer Behavior Dashboard

## 📌 Overview

This Data Analytics project analyzes customer purchasing behavior using **Python, MySQL, SQL, and Power BI**. The goal is to uncover customer trends, revenue patterns, and category performance through data analysis and interactive visualizations.

## 🛠️ Tech Stack

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* MySQL
* SQL
* Power BI

## 🚀 Workflow

* Data Loading & Cleaning using Python
* Exploratory Data Analysis (EDA)
* Data Storage in MySQL
* SQL Query Analysis
* Interactive Power BI Dashboard Creation

* Clothing generates the highest revenue and sales.
* 73% of customers are non-subscribers.
* Young adults contribute the highest revenue and sales.
* Customer behavior varies across categories and shipping methods.
